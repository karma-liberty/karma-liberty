มีรายงานของการเสียชีวิตอย่างเป็นทางการ และผลที่จะเกิดตามมาที่รุนแรงมาก หลังการได้รับการฉีดวัคซีน

นักวิทยาศาสตร์ที่มีชื่อเสียงหลายคน นักไวรัสวิทยา และแพทย์ในหลายๆประเทศ ออกมาเตือน ไม่ให้ผู้คนรับการฉีดวัคซีนที่มีส่วนประกอบของสารพันธุกรรมนี้ เนื่องจากพวกเขาได้ทำการพิสูจน์แล้วว่า เป็นอันตรายต่อระบบภูมิคุ้มกันตัวเองอย่างรุนแรงตามมา

นอกจากนี้ยังผิดกฎหมาย ภายใต้กฎหมายระหว่างประเทศ และกฏนูเรมเบิร์ก ในการทดลองทางการแพทย์เกี่ยวกับผู้คนโดยไม่ได้รับความยินยอมเป็นลายลักษณ์อักษร ซึ่งการจัดการฉีดวัคซีนให้กับประชาชนในครั้งนี้ เปรียบเสมือนเป็นการให้ผู้คนจำนวนมหาศาล เข้าร่วมการทดลอง และตามกฏการทดลองทางการแพทย์นั้น คนเหล่านั้นต้องได้รับรายละเอียดต่างๆ ได้รับความรู้และความเข้าใจอย่างเพียงพอหลังจากการให้คำปรึกษาที่ยาวนาน และได้รับการแจ้งให้ทราบอย่างเต็มที่เกี่ยวกับธรรมชาติ ระยะเวลาความเสี่ยง ปัญหาและอันตรายที่อาจเกิดขึ้น และประโยชน์ของการทดลอง!

(แน่นอนว่าไวรัสไม่ได้ทำให้เกิด "คลื่น" ที่ว่ากันว่าเกิดแล้วเกิดอีกอย่างที่มีการกล่าวอ้างอย่างต่อเนื่อง มันเป็นเพียงวิธีที่จะทำให้เกิดการแพร่ระบาดอย่างต่อเนื่องเพื่อสร้างความหวาดกลัว และบังคับให้ทุกคนเร่งรีบที่จะรับการฉีด นอกจากนี้การเสียชีวิตทั้งหมดที่เกิดจากการฉีดสารนี้ยังถูกกล่าวอ้างถึง "คลื่น" และ "การกลายพันธุ์" ใหม่เหล่านี้ ซึ่งเป็นเรื่องไร้สาระอย่างชัดเจน บรรดานักไวรัสวิทยาที่ซื่อสัตย์จะบอกคุณเช่นนั้น!)

หากคาดว่าผู้ที่ได้รับการฉีดจำนวนมากดังกล่าวจะเสียชีวิตหรือได้รับผลกระทบด้านสุขภาพอย่างรุนแรงจากการฉีดยาเหล่านี้ เหตุใดรัฐบาลจึงยังคงยืนยันที่จะบังคับให้ฉีดยาเหล่านี้กับประชากรของตน โดยการทำให้กลัว และในเชิงบีบบังคับ?
การฉีดสาร mRNAชนิดนี้ และอื่น ๆ เป็นการสร้างโปรแกรมใหม่ให้ร่างกายให้ผลิต Spike Protein ทำให้เกิดความเสียหายอย่างรุนแรงต่อหลอดเลือด และทำให้ผู้คนเสียชีวิตในเวลาต่อมา ส่วนใหญ่จะเกิดชัดเมื่อร่างกายได้รับเชื้อไวรัสตัวใหม่แล้วไม่สามารถสร้างภูมิคุ้มกันโดยธรรมชาติอย่างปกติด้วยตัวมันเองอีกต่อไป

ปัญหาที่แท้จริงและรุนแรงนี้ อาจจะเกิดขึ้นในอนาคตอันใกล้ บางทีใน 6 เดือนถึง 3 ปี หลังฉีดสารนี้ จะมีการอุบัติไวรัสใหม่ๆตามมา จะเห็นว่าสัตว์ทุกตัวในการทดลองตาย ด้วยภาวะแทรกซ้อนรุนแรง เนื่องจากพายุไซโตไคน์เกิดขึ้นกับระบบภูมิคุ้มกัน เข้าโจมตีอวัยวะของตัวเอง และเกิดภาวะแทรกซ้อนอื่น ๆ อีกมากมาย! ระลอกนี้จะถูกอ้างว่าเกิดอุบัติกลายพันธุ์ของไวรัสรุนแรงมากขึ้น มากขึ้น ความปกติสุขในการใช้ชีวิตของประชาชนในโลกนี้ก็ต้องเปลี่ยนไป ผู้คนต่างคนต่างอยู่ หวาดระแวงกันและกัน ไม่สามารถเดินทางไปพักผ่อนท่องเที่ยวอย่างอิสระ และผลกระทบอื่นๆไม่มีวันสิ้นสุด

(การเสียชีวิตเนื่องจากผลเหล่านี้ไม่สามารถกล่าวอ้างว่าเกิดจากการฉีดสารพันธุกรรมนั้นอย่างแน่นอน เพราะเกิดขึ้นภายหลังการฉีดชั่วระยะเวลาหนึ่ง ผู้ผลิตจึงไม่ต้องคำนึงหรือกล่าวถึงซึ่งจะมีผลกับการจำหน่ายเป็นอย่างมาก)

Spike Proteins เหล่านี้หรือ Prion บางรูปแบบกำลังถูกส่งจากคนที่ฉีดไปยังคนที่ไม่ได้ฉีด และทำให้เกิดปัญหาสุขภาพที่รุนแรงต่างๆในประชากรโดยรวม
นี่เป็นที่รู้จักกันดีเช่นกันว่า กระบวนการนี้เคยใช้ได้จริง ยกตัวอย่างในประเทศจีน เพื่อที่จะกำจัดฆ่าไก่ทั้งหมด พวกเขาทำการการฉีดวัคซีนไข้หวัดนกที่แพร่ระบาดขณะนั้นให้ไก่บางส่วน แล้วจากนั้นจะทำให้ไก่ที่ไม่ได้ฉีดวัคซีนติดเชื้อแล้วตายทั้งหมดไปด้วยกัน

และนี่ก็คือส่วนหนึ่งของแผนการ ที่จะฆ่าประชากรที่ไม่ได้รับการฉีดวัคซีนทั้งหมด ที่ต้องการมีสุขภาพที่ดี ที่ไม่ต้องการให้ดีเอ็นเอของพวกเขาถูกทำลาย แต่ต้องกลายเป็นสัตว์ประหลาดแฟรงเกนสไตน์ที่ป่วย และเป็นหมันจากการสัมผัสผู้ได้รับสารเหล่านี้หรือไม่... ซึ่งตอนนี้หมายความว่าผู้ที่ได้รับการฉีดจะต้องแยก และอยู่ห่างจากผู้ที่มีสุขภาพดีที่ไม่ได้รับการฉีดเพื่อให้พวกเขาปลอดภัย!

แพทย์ผู้ทรงคุณวุฒิ นักไวรัสวิทยา และนักวิทยาศาสตร์จำนวนมากได้ทำการวิเคราะห์ และต่างเห็นพ้องต้องกันว่า การฉีดสารเหล่านี้ทำไปเพื่ออะไร พวกเขาไม่มีเหตุผลอื่นใดนอกจากสรุปว่า การฉีดสารเหล่านี้ดูเหมือนจะเป็นโปรแกรมโดยเจตนาที่จะฆ่าคนจำนวนมาก

ไม่ต้องสงสัยเลยว่าการระบาดของโรค (plandemic) ทั้งหมดนี้เป็นแผนการลดจำนวนประชากร ซึ่งอยู่ในแผนการที่วางไว้มานานมากแล้ว โดยก่อนหน้านี้จะยกปัญหาทางธรรมชาติมาเป็นปัญหาหลัก แล้วหาทางแก้ไข โดยทุกครั้งจะมีการจำกัดชีวิตอิสระของผู้คน ทำให้เศรษฐกิจย่ำแย่ผู้คนตกงาน อดตาย สังคมปั่นป่วนวุ่นวาย แต่ยังทำได้ไม่สำเร็จ แต่ครั้งนี้หนักขึ้นมาก !!

มันคือการฆ่าล้างเผ่าพันธุ์ทั่วโลก Mass-Murder ในระดับที่ไม่สามารถจินตนาการได้เนื่องจากคนส่วนใหญ่จะเสียชีวิตจากการฉีดสารเหล่านี้ ตามที่ได้รับการยอมรับจาก SAGE( Scientific Advisory Group for Emergencies) และที่อื่น ๆ

► สิ่งที่ร้ายแรงที่สุดประการหนึ่งของอาชญากรรมที่ชั่วร้ายนี้คือ การทำลายยีนของคนที่ได้รับการฉีดเข้าไป ยีนของมนุษย์ทั้งหมดจะถูกเปลี่ยนแปลง และถูกทำลายอย่างถาวร ยิ่งไปกว่านั้นยีนที่ผิดปกตินี้สิ่งนี้จะถูกส่งต่อไปยังรุ่นต่อ ๆ ไปทั้งหมด ซึ่งจะทำให้เกิดข้อบกพร่องที่รุนแรง และสิ่งที่จะเกิดขึ้นยิ่งแย่กว่าที่เกิดขึ้นกับการใช้สารธาลิโดไมด์! ในอดีต
( สารชนิดหนึ่งที่สตรีตั้งครรภ์กินเข้าไปและเกิดความผิดปกติกับลูกที่เกิดมา แม้สารนั้นจะถูกยืนยันจากผู้ผลิตว่าปลอดภัย )
และน้อยคนนักที่จะมีชีวิตรอด หรือไม่เป็นหมันจากการฉีดสารเหล่านี้ หรือน้อยคนนักที่จะมีสุขภาพดีพอที่จะมีลูกได้

และนี่คืออุบัติการณ์การสูญพันธุ์ ซึ่งได้รับการวางแผนโดยผู้มีอำนาจปกครองระบอบเผด็จการมานานหลายศตวรรษ และกำลังดำเนินการอยู่ในขณะนี้ ด้วยความยินยอมของทุกรัฐบาลบนโลก!

♦ ทนายความหลายพันคน และนักวิทยาศาสตร์ที่กล้าหาญซื่อสัตย์ แพทย์และผู้เชี่ยวชาญด้านสุขภาพหลายหมื่นคนกำลังเตรียมการพิจารณาคดีนูเรมเบิร์กใหม่ สำหรับความผิดทั้งหมดของผู้ที่อยู่เบื้องหลังอาชญากรรมต่อมนุษยชาติครั้งใหญ่ที่สุดเท่าที่เคยมีมาครั้งนี้
โดยการแจ้งข้อหาเจ้าหน้าที่ของรัฐที่สมรู้ร่วมคิดทุกคน ,Bill Gates, Klaus Schwaab, Anthony Fauci, Tedros Adhanom, Deborah Birx เจ้าหน้าที่ UN และอื่น ๆ อีกมากมาย! เหล่านี้จะถูกนำมาไต่สวนดำเนินคดี อาชญากรรมต่อมวลมนุษยชาติ

♦ โปรดทราบว่ามีโทษประหารสำหรับการก่ออาชญากรรมต่อมนุษยชาติเหล่านี้!

ยังมีทางเลือกที่ปลอดภัย และมีประสิทธิภาพในการรักษาและป้องกันโรคซาร์ส - โควี –2 (โควิด 19) และเชื้อไวรัสต่างๆที่กล่าวอ้างว่ามีการกลายพันธุ์

การใช้วิตามินดี วิตามินซี สังกะสี Hydroxychloroquine (HCQ) และ ivermectin เป็นวิธีการรักษาและป้องกันโรคที่มีราคาถูก ปลอดภัย และมีประสิทธิภาพมากสำหรับการติดเชื้อไวรัสไข้หวัดและ SARS-CoV-2 แต่อย่างไรก็ตามตอนนี้พวกเขาถูกรังเกียจ ถูกแบน เพราะราคาถูก และมีประสิทธิภาพ ซึ่งเผยให้เห็นเรื่องราวผลสัมฤทธิ์เบื้องหลังการแพร่ระบาดครั้งนี้มากขึ้น แต่ถูกผู้มีอำนาจแบน!!

รัฐบาลกำลังละเมิดทุกรหัสเหล่านี้ซึ่งเป็นข้อเท็จจริงที่พิสูจน์ได้ง่าย:

  • Nuremberg Code # 1: ความยินยอมโดยสมัครใจเป็นสิ่งจำเป็น
  • Nuremberg Code # 2: ให้ผลลัพธ์ที่มีประสิทธิผลซึ่งไม่สามารถพิสูจน์ได้ด้วยวิธีการอื่น
  • Nuremberg Code # 3: :พื้นฐานของการทดลองต้องมาจากผลจากการทดลองในสัตว์ และประวัติโรคตามธรรมชาติ
  • Nuremberg Code # 4: หลีกเลี่ยงความทุกข์และการบาดเจ็บที่ไม่จำเป็นทั้งหมด
  • Nuremberg Code # 5: ไม่ต้องทำการทดลองหากมีเหตุผลที่จะคิดว่าจะมีการบาดเจ็บหรือเสียชีวิต
  • Nuremberg Code # 6: ความเสี่ยงไม่ควรมาก่อนผลดีที่ผู้คนจะได้รับ
  • Nuremberg Code # 7: ต้องเตรียมรับมือแม้กระทั่งความเป็นไปได้ที่จะบาดเจ็บพิการหรือเสียชีวิตจากความเสี่ยงทั้งหลายแม้จะเล็กน้อยก็ตาม
  • Nuremberg Code # 8: การทดลองต้องดำเนินการโดยบุคคลที่มีคุณสมบัติทางวิทยาศาสตร์
  • Nuremberg Code # 9: ทุกคนต้องมีอิสระในการทำให้การทดลองสิ้นสุดลงได้ทุกเมื่อ
  • Nuremberg Code # 10: นักวิทยาศาสตร์ต้องทำให้การทดลองสิ้นสุดลงเมื่อใดก็ได้หากมีสาเหตุที่เป็นไปได้ที่จะส่งผลให้เกิดการบาดเจ็บหรือเสียชีวิต

บทความในหนังสือพิมพ์ยอมรับอย่างเปิดเผยว่า 75% ของการเสียชีวิตเกิดจากการฉีดสารเหล่านี้ เช่นเดียวกับในประเทศอื่น ๆ สถิติในอินเดียพบว่าคลื่นลูกที่สองหรือสามเกิดจากการฉีดสารนี้ อินเดีย: Ivermectin ลดการเสียชีวิตของ Covid จากนั้นการฉีดเริ่มฆ่าตัวเลขจำนวนมาก
